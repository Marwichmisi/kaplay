# KAPLAY Game Development Skill

Créer des jeux 2D avec **KAPLAY.js v4000** (successeur de Kaboom.js).

## Installation

```bash
npx skills add Marwichmisi/kaplay --skill kaplay
```

## Contenu du skill

- `SKILL.md` — instructions principales
- `references/` — 12 fichiers de référence par domaine (physique, rendu, entrées, niveaux, etc.)
- `evals/` — cas de test pour l'évaluation du skill
